Current Project:
IonSolver is a magnetohydrodynamic simulation software utilising the Lattice-Boltzmann-Model for fluid interaction. The software is written in the rust programming language and uses OpenCL integration for parallelization. Most kernel code is taken with permission from FluidX3D, the simulation setup, rendering etc. are all written by us. It aims to provide a physically accurate and very fast simulation, with support for solid-fluid interaction and multiple boundary conditions. You can find it on GitHub.
